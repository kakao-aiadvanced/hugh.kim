## 시작하기

프로젝트를 시작하려면 다음 단계를 따라주세요:

1. 저장소를 클론합니다
```bash
git clone https://github.com/kakao-aiadvanced/hugh.kim.git
```

2. 프로젝트 디렉토리로 이동합니다
```bash
cd hugh.kim
```
